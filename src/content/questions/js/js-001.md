## 🧩 题目

请解释 JavaScript Event Loop 的执行机制。

---

## 💡 解题思路

1. JS 是单线程，所有任务都在同一个主线程上执行
2. 同步任务直接在 Call Stack 中执行
3. 异步任务完成后会被放入 MacroTask / MicroTask 队列
4. 每一轮循环：先清空所有 microtask，再取一个 macrotask

---

## ✅ 面试回答（标准版）

vent Loop 是 JavaScript 处理异步代码的核心机制。JavaScript 是单线程的，但为了能处理多个任务，它使用了事件循环来管理同步任务和异步任务的执行。执行栈会先执行同步任务，若栈为空，事件循环会检查消息队列，并将异步任务执行。微任务队列中的任务优先于宏任务执行，因此 Promise 的回调会先于 setTimeout 的回调执行。通过这种方式，JavaScript 能够实现非阻塞的异步操作。
