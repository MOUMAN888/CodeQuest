## 🧩 题目

ES6（ES2015）有哪些比较重要的新特性？请简要列举并说明其作用。

---

## 💡 解题思路

1. 变量与作用域：`let` / `const`、块级作用域
2. 函数相关：箭头函数、默认参数、rest 参数
3. 解构赋值、扩展运算符
4. 新增类型与内置对象：`Symbol`、`Map/Set`、`Promise`
5. 模块化、`class`、模板字符串等

---

## ✅ 面试回答（标准版）

ES6（更正式地叫 ES2015）是 JavaScript 语言发展中的一个重要版本，引入了大量语言级的新特性，极大改善了代码的**可读性、可维护性和表达能力**。常见的重要特性包括：

1. **块级作用域与新变量声明**  
   - `let`：块级作用域的变量声明，解决了 `var` 的变量提升和全局污染问题  
   - `const`：声明只读常量（绑定不可变，但引用类型内部仍可修改），同样具有块级作用域  
   - 块级作用域：`if`、`for` 等语句块可以形成独立的作用域  

2. **函数相关增强**  
   - 箭头函数：更简洁的函数写法 + 词法 `this` 绑定  
   - 默认参数：`function foo(a = 1) {}`，避免手动处理 `a = a || 1` 等写法  
   - rest 参数：`function sum(...args) {}`，替代 `arguments`，更直观  

3. **解构赋值与扩展运算符**  
   - 数组解构：`const [a, b] = [1, 2];`  
   - 对象解构：`const { x, y } = point;`  
   - 扩展运算符：`...` 用于数组/对象的展开与合并，如 `[...arr1, ...arr2]`、`{ ...obj, a: 1 }`  

4. **新增数据类型与内置数据结构**  
   - `Symbol`：新的原始数据类型，用于创建唯一的标识符，常用作对象私有属性键  
   - `Map` / `Set`：更适合映射与集合的专用数据结构  
   - `WeakMap` / `WeakSet`：弱引用集合，利于垃圾回收  

5. **Promise 与异步编程基础**  
   - `Promise`：提供比回调函数更优雅的异步流程控制方式，是后续 `async/await` 的基础  
   - 链式调用、错误捕获（`catch`）、多个异步任务组合（`Promise.all` / `race`）等  

6. **模块化与类（Class）**  
   - ES Module：`import` / `export`，提供标准化的模块系统，替代早期的 CommonJS/AMD 等方案（在浏览器和打包工具中得到广泛支持）  
   - `class`：基于原型的语法糖，提供更接近面向对象语言的类、继承（`extends`）、构造器和静态方法写法  

7. **模板字符串与其他语法糖**  
   - 模板字符串（Template Literals）：使用反引号 `` ` ``，支持多行字符串和 `${}` 插值  
   - 对象字面量增强：属性名简写、方法简写、计算属性名等  
   - `for...of` 循环：简化对可迭代对象（数组、字符串、Map、Set 等）的遍历  

综上，ES6 的这些特性共同让 JavaScript 更接近现代语言的开发体验，大大提升了代码的表达力与工程化能力，也是当前前端开发的基础语法标准。
