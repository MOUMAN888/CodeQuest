## 🧩 题目

说说你对媒体查询（Media Query）的理解？它的语法形式有哪些，通常用来解决什么问题？

---

## 💡 解题思路

1. 定义：根据设备或视口特性有条件地应用样式
2. 常见媒体特性：宽高（`width/height`、`min-width` 等）、分辨率、方向（`orientation`）等
3. 书写方式：`@media` 规则、`link` 标签的 `media` 属性
4. 常见应用：响应式布局、移动端适配、暗黑模式、打印样式等

---

## ✅ 面试回答（标准版）

媒体查询（Media Query）是 CSS 中实现**响应式设计**的核心能力之一，它允许我们根据设备或视口的不同特性（如宽度、高度、分辨率、方向等）有条件地应用不同的样式，从而在手机、平板、PC 等不同环境下都能获得合适的展示效果。

1. **基本语法形式**  
   最常见的是使用 `@media` 规则包裹一段样式：

   ```css
   @media (max-width: 768px) {
     .sidebar {
       display: none;
     }
   }
   ```

   表示当视口宽度小于等于 768px 时，隐藏 `.sidebar`。  
   也可以在 `<link>` 标签上使用 `media` 属性：

   ```html
   <link rel="stylesheet" href="print.css" media="print" />
   ```

2. **常见媒体特性**  
   - 尺寸相关：`width`、`height`、`min-width`、`max-width` 等（最常用）  
   - 设备像素比：`resolution`（例如 Retina 屏的图片处理）  
   - 方向：`orientation: portrait | landscape`（横屏/竖屏）  
   - 媒体类型：`screen`、`print` 等  
   - 现代特性：如 `prefers-color-scheme` 检测浅色/深色主题偏好  

   可以通过 `and`、`,`（or）、`not` 等进行组合：

   ```css
   @media screen and (min-width: 1024px) {
     /* PC 端样式 */
   }
   ```

3. **典型应用场景**  
   - **响应式布局**：根据不同屏幕宽度调整布局（如从多列变单列、调整字体大小和间距）  
   - **移动端适配**：在小屏设备上隐藏不必要的信息，或增大可点击区域  
   - **打印样式**：使用 `media="print"` 或 `@media print` 为打印提供专门的样式  
   - **暗黑模式**：通过 `@media (prefers-color-scheme: dark)` 为系统深色模式提供适配样式  

总体来说，媒体查询是 CSS 中用来**根据环境做分支逻辑**的机制，是实现响应式 Web 设计（RWD）的基础工具之一，合理规划断点和样式结构能够极大提升多终端的体验和维护性。
