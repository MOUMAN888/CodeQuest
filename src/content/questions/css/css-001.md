## 🧩 题目

请解释 CSS 盒模型（Box Model），并说明 `box-sizing: border-box` 与默认值的区别。

---

## 💡 解题思路

1. 盒模型由 content、padding、border、margin 组成
2. 默认 `content-box`：width 只包含内容区
3. `border-box`：width 包含内容 + padding + border

---

## ✅ 面试回答（标准版）

在 CSS 中，每一个元素都可以看成一个矩形盒子，由内容区域（content）、内边距（padding）、边框（border）和外边距（margin）组成，这就是盒模型。默认的 `box-sizing: content-box` 表示设置的 width/height 只作用于内容区域，padding 和 border 会额外把盒子撑大。而 `box-sizing: border-box` 表示 width/height 包含内容、padding 和 border，在布局时更好算尺寸，能够避免频繁手动减 padding/border。
