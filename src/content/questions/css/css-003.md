## 🧩 题目

有哪些常见的 CSS 选择器？请分别介绍它们的含义与使用场景。

---

## 💡 解题思路

1. 基本选择器：元素选择器、类选择器、ID 选择器、通配选择器
2. 组合选择器：后代、子代、相邻兄弟、普通兄弟
3. 属性选择器：按属性名、属性值、前缀、后缀、包含匹配
4. 伪类选择器：状态伪类（`:hover`、`:focus`）、结构伪类（`:first-child` 等）
5. 伪元素选择器：`::before`、`::after` 等生成内容

---

## ✅ 面试回答（标准版）

CSS 选择器用来「选中页面上的某些元素」，再对其应用样式。常见的选择器大致可以分为以下几类：

1. **基本选择器**  
   - **元素选择器**：如 `div`、`p`，按标签名选中所有该类型元素  
   - **类选择器**：如 `.btn`，按 `class` 选中，可复用性强，是布局和组件开发中最常用的  
   - **ID 选择器**：如 `#header`，按 `id` 选中，理论上在页面中唯一，一般用于锚点或 JS 钩子，样式中不建议大量依赖  
   - **通配选择器**：`*`，选中所有元素，常用于初始化（如 `* { box-sizing: border-box; }`），要注意性能和影响范围  

2. **组合选择器**  
   - **后代选择器**：`A B`，表示 B 是 A 的任意层级后代，如 `.nav a`  
   - **子代选择器**：`A > B`，表示 B 是 A 的直接子元素，如 `.list > li`  
   - **相邻兄弟选择器**：`A + B`，表示紧挨着 A 之后的第一个兄弟 B，如 `h1 + p`  
   - **普通兄弟选择器**：`A ~ B`，表示 A 之后的所有兄弟 B，如 `.title ~ p`  

3. **属性选择器**  
   - `[attr]`：选中拥有某个属性的元素，如 `[disabled]`  
   - `[attr="val"]`：属性值等于某个值，如 `input[type="text"]`  
   - `[attr^="val"]`：属性值以 `val` 开头，如 `a[href^="https"]`  
   - `[attr$="val"]`：属性值以 `val` 结尾，如 `img[src$=".png"]`  
   - `[attr*="val"]`：属性值包含 `val`，如 `[class*="btn-"]`  

4. **伪类选择器**（pseudo-classes）  
   - **状态伪类**：`:hover`、`:active`、`:focus`、`:visited` 等，表示元素的交互/状态  
   - **结构伪类**：`:first-child`、`:last-child`、`:nth-child(n)`、`:nth-of-type(n)` 等，用于按结构位置选择  
   - **其他常用伪类**：`:not(selector)` 用于排除匹配；`:checked` 用于选中状态的表单控件等  

5. **伪元素选择器**（pseudo-elements）  
   - 常见的有 `::before`、`::after`、`::first-line`、`::first-letter` 等  
   - 其中 `::before` 和 `::after` 最常用，配合 `content` 属性可以在不改变 DOM 结构的情况下插入装饰性内容，用于图标、小三角、分隔线等  

在真实项目中，通常会**优先使用类选择器**来做样式命名，搭配少量结构伪类和伪元素实现复杂效果，同时避免过于复杂、层级过深的选择器，以保证样式的可维护性和性能。
